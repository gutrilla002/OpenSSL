=head1 NAME

ossl_print_gens - print GeneralNames in a human-friendly, multi-line string

=head1 SYNOPSIS

 #include <openssl/x509v3.h>

 int ossl_print_gens(BIO *out, GENERAL_NAMES *gens, int indent);

=head1 DESCRIPTION

ossl_print_gens() prints a human readable version of the GeneralNames I<gens> to
BIO I<out>. Each line is indented by I<indent> spaces.

=head1 RETURN VALUES

ossl_print_gens() always returns 1.
